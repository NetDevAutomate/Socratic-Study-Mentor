## Architecture Notes

### Eval flow (current)
```
studyctl eval run
  → PersonaTarget.setup() — builds canonical persona, writes to temp file
  → PersonaTarget.run() — claude -p with --append-system-prompt-file
  → LLMJudge.score() — heuristic pre-flight → Bedrock Sonnet 4.6 rubric
  → JudgeResult with dimensions + feedback
  → TSVReporter.record() → eval-results.tsv
  → Markdown report → docs/reports/
```

### Key files
- `packages/studyctl/src/studyctl/eval/targets/persona.py` — claude -p based eval target
- `packages/studyctl/src/studyctl/eval/judge/llm.py` — heuristic + LLM rubric judge
- `packages/studyctl/src/studyctl/eval/llm_client.py` — Ollama/OpenAI/Bedrock client
- `packages/studyctl/src/studyctl/eval/prompts/judge-rubric.md` — 7-dimension rubric + feedback
- `packages/studyctl/src/studyctl/eval/orchestrator.py` — scenario loop
- `packages/studyctl/src/studyctl/cli/_eval.py` — CLI commands (run, setup, history)
